# SQL Course Progress – Udemy

Technical repository documenting my learning progress through the Udemy course **“The Complete SQL Bootcamp (30 Hours): Go from Zero to Hero”** by **Baraa Khatib Salkini**.

This repository includes structured notes, exercises, and SQL scripts developed while following the course. All exercises are derived from the course content and executed using **Microsoft SQL Server** and **SQL Server Management Studio (SSMS)**.
